# OmNom.js - An (Almost) Universal BNF Parser

 Lightweight Universal BNF Parser, pure JavaScript, ~300 LOC

## Use
 var parser = BNFParser(32, 'Your BNF grammar');  
 var tree = parser('Text to parse');
